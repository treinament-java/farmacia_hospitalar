# Farmácia Hospitalar

Projeto desenvolvido para gerenciamento de medicamentos em ambiente hospitalar.

## Funcionalidades

- Cadastro de medicamentos
- Controle de estoque
- Registro de entrada e saída
- Histórico de movimentações
- Integração com API REST

## Tecnologias utilizadas

- Java
- Spring Boot
- PostgreSQL
- Swagger
- FlutterFlow
- DBeaver

## Estrutura do sistema

O sistema possui três entidades principais:

- Medicamento
- Gerenciamento
- Histórico

## Objetivo

Auxiliar no controle de medicamentos hospitalares, organizando estoque, movimentações e registros de utilização.
